# Personal
项目名称：Personal
项目功能：1.通过个人主页介绍自己
         2.响应式页面，通过改变窗口大小，页面也会有所改变
         3.点击右下角箭头可回到顶层
         4.点击顶层任务栏可下滑到相应位置
         5.鼠标放在图片上可显示图片介绍
         等功能
使用方法：通过javascript、html5、css3、bootstrap等代码，为画面添加很多效果，使其更加美观
